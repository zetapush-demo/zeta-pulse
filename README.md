# Zetapulse - v1

## Basic angular example of a zetapush project with group and messaging services

#### Installation

`yarn install`

---

#### Start local worker

`yarn start-worker`

---

#### Start local front

`yarn start-front`

---

#### Project Structure

```

│
└─── front
```
